벌써 인턴 생활 3일차 ~

현재 분위기나 기분을 보면 일주일은 지난 것 같은데 아직 3일차인게 놀랍다.
HTML 학습 진행상황은 기본적인 태그 사용법과
외부 페이지 및 내부 html 링크 연결까지 마무리했다.
어제인 7월 2일에 쓴 글은 여기에 링크 걸어봤다.

<a href="1.html" target="_blank">여기</a>
링크를 걸기 위해선 a 태그를 사용한다. 속성으로는 href에 넘어갈 주소를 주면 된다.
기본값으로 같은 창에서 페이지만 넘어가는데 target에 "_blank"를 주면 새탭을 열어 해당 페이지로 넘어간다.
